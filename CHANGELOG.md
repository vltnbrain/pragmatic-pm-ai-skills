# Changelog

All notable changes to this project are documented in this file.

The format is based on Keep a Changelog and this project follows Semantic Versioning.

## [1.1.0] - 2026-04-20

### Added
- Added `CHANGELOG.md` for versioned release notes.
- Added `BENCHMARK.md` with a practical before/after measurement framework.
- Expanded `EXAMPLES.md` to 16 pragmatic cases with "bad vs pragmatic" patterns.

### Changed
- Updated `README.md` to include benchmark and release artifacts in repository structure.
