# Pragmatic PM AI Skills

面向产品经理、IT 从业者和职业发展场景的务实型 AI 指南。

本仓库参考了 [forrestchang/andrej-karpathy-skills](https://github.com/forrestchang/andrej-karpathy-skills) 的结构，但核心目标是让 AI 在 PM 与职业决策中避免空话，直接产出可执行结果。

## 核心问题

通用 AI 输出常见问题：
- 文本很长，但无人阅读；
- 框架很多，但没有决策价值；
- 建议听起来正确，却忽略真实约束（时间、预算、权限、注意力）。

## 解决方案

本仓库提供可复用规则，要求 AI：
- 先做激励审计，再给建议；
- 先降摩擦，再加流程；
- 用硬信号替代叙事噪声；
- 用真实行为变化验证结果。
